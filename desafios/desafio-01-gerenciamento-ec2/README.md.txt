# 💻 Desafio 01 – Gerenciamento de EC2 e Serviços AWS

## 🧠 Contexto

Uma empresa fictícia é especializada na criação de **cenários para RPG** e precisa criar uma arquitetura na **AWS** que suporte o sistema principal, armazenamento de arquivos e processamento automatizado.

---

## 🎯 Objetivos do Desafio

- Criar uma instância **EC2** para hospedar o sistema principal  
- Configurar um volume **EBS** como armazenamento persistente  
- Utilizar o **S3** para armazenar objetos (imagens, mapas, cenários)  
- Implementar uma função **Lambda** acionada por eventos no S3
